## 第三方组件清单

| 组件 | 用途 | 来源 | 许可 | 说明 |
|---|---|---|---|---|
| StarryOS / tgoskits（rcore-os） | 操作系统内核 + RK3588 rknpu 驱动 + 板级支持（**基础版本**） | https://github.com/rcore-os/tgoskits | Apache-2.0 | 我们基于其编译 Orange Pi 5 Plus 内核并启用 `rknpu`；增量贡献见 `基础版本与增量贡献.md` |
| rknn-toolkit2（airockchip） | 服务器侧 ONNX→RKNN 模型转换与量化、模拟器评测 | https://github.com/airockchip/rknn-toolkit2 | Apache-2.0（含部分专有运行库） | 仅在 x86 服务器侧使用，未修改其源码 |
| librknnrt.so（Rockchip RKNPU Runtime） | 板上 NPU 推理运行时（被 `act-rknn` 动态链接） | Rockchip 官方 SDK / rknn-toolkit2 仓库 | Rockchip 专有（闭源），允许随应用分发 | 以二进制形式放入镜像 rootfs；版本 2.3.2 |
| RK3588 idbloader + u-boot.itb（板载引导块） | 自烧整盘镜像的一二级引导 | 从 Orange Pi 5 Plus 出厂/官方系统的存储中 `dd` 提取 | U-Boot 部分为 GPL-2.0；DDR init/loader（rkbin）为 Rockchip 专有，允许再分发 | ⚠️ 该引导块以二进制内嵌于我们发布的 Release 镜像并分发。U-Boot 源码可从 Orange Pi / Radxa 官方 u-boot 仓库获得；本队未修改引导块 |
| bootlin aarch64 glibc 运行时 / 工具链 | rootfs 内 glibc 动态运行时、交叉编译工具链 | https://toolchains.bootlin.com | glibc：LGPL-2.1+/GPL；工具链各组件各自许可 | 以运行时库形式注入 rootfs |
| 官方 musl rootfs（StarryOS 配套） | rootfs 基底 | tgoskits / StarryOS 配套 | 随上游 | 在其上注入 glibc 与推理载荷 |
| ACT 模型 / proj57 数据集 | 推理对象（模型权重）与评测帧 | 赛题提供；数据集 HuggingFace `bobodai/proj57_dataset` | 见赛题/数据集声明 | 仅用于本赛题推理与评测 |
| ONNX Runtime（任务三） | 任务三推理引擎之一 | https://github.com/microsoft/onnxruntime | MIT | 任务三静态链接使用（详见任务三分支） |
| tract（任务三） | 任务三纯 Rust 推理引擎 | https://github.com/sonos/tract | MIT / Apache-2.0 | 任务三使用 |

> 说明：表中 `librknnrt.so`、`idbloader`、`rkbin` 为 Rockchip 专有二进制，按其随附许可允许随应用、设备分发；我们仅以二进制形式集成，未做反编译或修改。U-Boot 遵循 GPL-2.0，对应源码可从其上游公开仓库获取，本队未修改。
